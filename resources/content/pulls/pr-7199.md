Resolves #7195 
I added an example component for testing the OpenStreetMaps addon. It will be a functional copy of the GoogleMaps example component. This PR implements create() and panTo() in the addon. The example component loads the map, supports zooming in and out, and panning between Iceland and San Francisco.
