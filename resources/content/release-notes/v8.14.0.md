# Enhanced Grid Column Header Drag&Drop

* We can now optionally move entire columns
* `component.Base`: `executeVdomUpdate()`, `update()`, `promiseUpdate()` => removed the vdom & vnode params
* prevent vnodes from getting cloned
* delay the creation of record classes
* `wrapperCls` updates will now always use the vdom engine
