# Refine DevIndex Introduction Disclaimer and Cap Context

The current legal disclaimer in the DevIndex `Introduction.md` refers to the project as a "tech demo" and uses marketing language. We need to rewrite this to adopt an "understatement" philosophy, framing it as an independent community resource.

Additionally, we need to clarify the context around the 50k user cap in the documentation, explaining that while the engine can effortlessly handle more, the cut is made purely to limit the download size (currently ~23MB before gzip) and mentioning the current entry threshold (2,134 lifetime contributions as of Feb 27, 2026).
