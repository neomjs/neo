### Architectural Summary
This Pull Request modernizes the AI infrastructure documentation within the core codebase overview. It updates the underlying topological model and properly demarcates the Neo.mjs Frontend Engine from the Node.js Agent SDK.

### Fat Ticket Details
- **Expansion of MCP Server Scale**: Replaced the legacy 3-server topology with the current 5-server topology, explicitly including the `neural-link` and `file-system` servers.
- **Architectural Boundary Decoupling**: Codified the absolute isolation between the browser-based presentation layer and the Node.js Agent OS, specifying JSON-RPC via Neural Link as the solitary bridge.
- **Whitebox E2E Documentation**: Introduced the deterministic Whitebox E2E capabilities directly into the Neural Link server overview.
- **DreamService Advancements**: Documented the shift towards Hybrid RAG pipelines, SQLite-backed Native Graph DB, and local orchestration (e.g. `gemma4-31b`).

These additions ensure that the codebase index is structurally aligned with the active Agent OS capabilities and maintains object-permanent consistency across future R&D iterations.
