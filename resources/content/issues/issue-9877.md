# Docs: Modernize Historical Context and Sandbox topology in CodebaseOverview

### Description
Following the recent AI infrastructure updates, the `Historical Context` section in `CodebaseOverview.md` remains pointed at deprecated `.github` directories and is missing critical context categories.

### Proposed Changes
- **Directory Redirection:** Update the Historical Context paths to accurately reflect the `resources/content/*` directory structure.
- **Pull Requests & Sandbox:** Formally document the ingestion of Pull Requests (`pulls/`) and GitHub Discussions (`discussions/` - Ideation Sandbox) into the Agent OS sync pipeline.
- **Workflow Enhancements:** Update the "When Stuck" troubleshooting section to strategically recommend querying these new Discussion and PR pipelines for historical architectural rationale.
