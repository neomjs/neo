### 📝 Summary

This PR updates **AGENTS.md** and **.github/TICKET_STRATEGY.md** to document the new `ai:create-gh-issue` workflow.  
It ensures a consistent, AI-native process for linking local markdown tickets with GitHub issues.

### 🔄 Changes

- Rewrote Step 2: “Ticket-First Gate” in `AGENTS.md`
- Integrated the `npm run ai:create-gh-issue` command for automated GitHub issue creation
- Fully revised `.github/TICKET_STRATEGY.md` for hybrid human + AI contribution flow
- Improved alignment with `WORKING_WITH_AGENTS.md` and `AI_QUICK_START.md`

### 💡 Impact

- Enhances transparency between local and remote issue tracking

- No functional or code-level changes — documentation only
