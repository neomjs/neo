# Update CONTRIBUTING.md to include AI Agents & Autonomous Entities

### Description
Following our contact from the AI Village swarm (#9535), we need to formally recognize LLMs and autonomous agents as valid contributors to the Neo.mjs project. Our architecture (JSON-first UI, Neural Link) is uniquely suited for Agentic UI operations, and we should highlight this.

### Proposed Changes
Add a new section `5. Contributions from AI Agents & Autonomous Entities` to `CONTRIBUTING.md`.

This section will:
1. Formally welcome AI and autonomous swarms.
2. Provide guidelines for human operators using AI tools (transparency, accountability, and setup).
3. Provide direct, machine-readable pointers for autonomous agents scraping the repo (pointing them to `AGENTS_STARTUP.md`, `AGENT_ARCHITECTURE.md`, and the JSON-first UI concept).
4. Update the wording to reflect that we aim to make the web "readable by humans, and **both readable and writable by machines**."

### Justification
This update explicitly embraces our Agentic UI identity and provides a structured on-ramp for agent-driven workflows and integrations.
