# [Handoff Protocol] Agent Startup Reconciliation & Graph Mutation

### Description
Agents must autonomously consume and resolve topological handoffs generated by the Sandman REM process. This task ensures `AGENTS_STARTUP.md` explicitly mandates checking `resources/content/sandman_handoff.md` and standardizes the graph mutations required to reconcile alert conflicts (e.g., manually closing an issue marked as SUPERSEDED by the graph).

### Acceptance Criteria
- Require agents to read `sandman_handoff.md` via tools during startup.
- Establish Native Edge Graph update procedures for conflict resolution.
- Related to Epic #9673.
